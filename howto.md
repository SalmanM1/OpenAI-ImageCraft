
# OpenAI Image Creator

This application is designed to create images using OpenAI's Dall-E models, utilizing Node.js and the Express framework.

![App Preview](public/img/screen.png)

## Getting Started

1. **Rename the Environment File**: Change the name of `example.env` to `.env`.
2. **Obtain an API Key**: Register at [OpenAI](https://beta.openai.com/) and add the API key to the `.env` file.
3. **Install Dependencies**: Run the following command to install the required packages:
   ```bash
   npm install
   ```
4. **Start the Server**: Use the following command to start the server:
   ```bash
   npm run run
   ```
5. **Access the Application**: Visit `http://localhost:5000` in your browser to use the image creator.

## Development Mode

To run the server in development mode, use the following command:
```bash
npm run develop
```
